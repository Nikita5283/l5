Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий. Приложение должно выполнять следующие функции:
1. Чтение базы данных из excel файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно и отразить в
отчёте, можно использовать запрос, данный в примере):
    - 1 запрос с обращением к одной таблице
    - 1 запрос с обращением к двум таблицам
    - 2 запроса с обращением к трем таблицам


    2 запроса должны возвращать перечень, 2 запроса одно значение.<br>

6. Сохранение изменений в excel файле.<br>


Элементами базы данных являются объекты классов согласно вашему варианту.<br>
Содержание классов определить самостоятельно и отразить в отчете (в классах должны присутствовать свойства,
конструкторы, перегруженный метод ToString).<br>
Весь функционал приложения реализовать в виде
методов вспомогательного класса с помощью LINQ-запросов.<br>
Предусмотреть обработку возможных ошибок при работе программы.<br>
