## Формулировка

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий. Приложение должно выполнять следующие функции:
1. Чтение базы данных из excel файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно и отразить в
отчёте, можно использовать запрос, данный в примере):
    - 1 запрос с обращением к одной таблице
    - 1 запрос с обращением к двум таблицам
    - 2 запроса с обращением к трем таблицам


    2 запроса должны возвращать перечень, 2 запроса одно значение.<br>

6. Сохранение изменений в excel файле.<br>


Элементами базы данных являются объекты классов согласно вашему варианту.<br>
Содержание классов определить самостоятельно и отразить в отчете (в классах должны присутствовать свойства,
конструкторы, перегруженный метод ToString).<br>
Весь функционал приложения реализовать в виде
методов вспомогательного класса с помощью LINQ-запросов.<br>
Предусмотреть обработку возможных ошибок при работе программы.<br>

В файле LR5-var9.xls приведён фрагмент базы данных «Гостиница».<br>
База данных состоит из трёх таблиц.<br>
Таблица «Клиенты» содержит данные о клиентах: ФИО и место жительства.<br>
Таблица «Бронирование» содержит информацию о зарегистрированных заявках на бронирование номеров:<br>
код клиента, код номера, номер бронирования, дата бронирования, дата заезда, дата выезда.<br>
Таблица «Номера» содержит информацию о забронированных номерах гостиницы:<br>
код номера, номер комнаты, этаж, число мест, стоимость проживания за сутки, категория гостиницы.<br>

## Алгоритм решения

1. Создать класс Booking для записи объектов из таблицы "Бронирование"

    1. Поля:
        - Код бронирования
        - Код клиента
        - Код номера
        - Дата бронирования
        - Дата заезда
        - Дата выезда

    1. Конструктор со всеми вышеперечисленными параметрами
    1. Перегрузка метода ToString()

1. Создать класс Client для записи объектов из таблицы "Клиенты".
    1. Поля:
        - Код клиента
        - Фамилия
        - Имя
        - Отчество
        - Место жительства

    1. Конструктор со всеми вышеперечисленными параметрами
    1. Перегрузка метода ToString()
    
1. Создать класс Room для записи объектов из таблицы "Номера".
    1. Поля:
        - Код номера
        - Этаж
        - Число мест
        - Стоимость проживания
        - Категория

    1. Конструктор со всеми вышеперечисленными параметрами
    1. Перегрузка метода ToString()

1. Создать вспомогательный класс HotelService для управления базой данной excel
    1. В качестве полей создать 3 списка
        - Clients: для хранения клиентов
        - Rooms: для хранения номеров
        - Bookings: для хранения записей бронирования

    1. Методы
        - LoadFromExcel: Из каждого листа эксель файла читает данные построчно и записывает их в качестве объектов соответствующих классов
        - GetDateFromCell: Чтение даты из ячейки эксель файла
        - SaveToExcel: Из каждого списка построчно записывает данные в соответствующий лист эксель файла
        - WriteHeader: Создает заголовки для каждого листа
        - ViewClients: Выводит на консоль всех клиентов
        - ViewBooking: Выводит на консоль все бронирования
        - ViewRooms: Выводит на консоль все номера
        - AddClient: Добавляет нового клиента в список клиентов
        - AddRoom: Добавляет новый номер в список номеров
        - AddBooking: Добавляет новую запись бронирования в список бронирований
        - DeleteClientById: Удаляет клиента по коду клиента
        - DeleteRoomById: Удаляет номер по коду номера
        - DeleteBookingById: Удаляет запись бронирования по коду бронирования
        - GetRoomsByCategory: Возвращает перечень номеров заданной категории (1 запрос к 1 таблице)
        - GetBookingsCountByClientId: Возвращает количество бронирований для заданного кода клиента (1 запрос к 2 таблицам)
        - GetBookingsDetailed: Возвращает подробный перечень бронирований с информацией о клиенте и номере (1 запрос к 3 таблицам)
        - GetTotalRevenue: Возвращает общую выручку от всех бронирований (1 запрос к 3 таблицам)
